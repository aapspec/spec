# Contributing to AAP

Thank you for your interest in contributing to the Agent Authorization Profile!

## Ways to Contribute

1. **Provide Feedback on Specification**
   - Review the draft
   - Comment on IETF mailing list: oauth@ietf.org
   - Open GitHub issues for technical clarifications

2. **Implement AAP**
   - Build Authorization Servers or Resource Servers
   - Share implementation experiences
   - Contribute test vectors

3. **Improve Documentation**
   - Fix typos or unclear sections
   - Add examples
   - Translate documentation

## Submission Process

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request
